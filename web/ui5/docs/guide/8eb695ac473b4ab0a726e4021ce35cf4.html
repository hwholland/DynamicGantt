Editing Status Filter

The editing status filter is enabled by default in the smart template for the list report.

If desired, you can disable this filter for your app. To do so, set sap:filterable to false for the navigation property of the DraftAdministrativeData as shown below:
#!xml<NavigationProperty xmlns:sap="http://www.sap.com/Protocols/SAPData" Name="DraftAdministrativeData" Relationship="draftinfrastructure.SalesOrderToDraftAdministrativeDataType
"FromRole="FromRole_SalesOrderToDraftAdministrativeDataType" ToRole="ToRole_SalesOrderToDraftAdministrativeDataType" sap:filterable="false"/>
