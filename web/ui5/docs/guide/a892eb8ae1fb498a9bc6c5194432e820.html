Extending Generated Apps

You can extend your smart templates application if needed. The smart templates extension concept follows the SAPUI5 extension concept. For more information, see Extending Apps.
Caution Extensions should be used with caution and only if no other ways, for example manifest settings or annotations, can be used to achieve the required behaviour.
You have the following options to extend your application:
  • Redefine a whole page by pointing to a different component in the app descriptor file manifest.json

    For more information, see Pointing to a Different Component in the App Descriptor.

  • Add additional sections or replace existing sections on the object page

    For more information, see Extension Points for Facets.

  • Extend the smart filter bar by adding custom fields

    For more information, see Custom Fields in the Smart Filter Bar.
