Post-Generation Steps

The following information details actions you can take after your app has been generated.

  • Following the generation of the app, it is mandatory to maintain facet texts.

    For more information, see Maintaining Facet Texts.

  • In addition, you have several options to manually enhance the generated app.

    For more information, see:

    • Setting Up General Features
    • Adapting the List Report
    • Setting Up Object Pages
    Note These enhancements are not modifications. They are overwritten if the app is regenerated, which occurs if you restart the SAP Web IDE wizard for the same app. The wizard is only intended for one-time generation.
