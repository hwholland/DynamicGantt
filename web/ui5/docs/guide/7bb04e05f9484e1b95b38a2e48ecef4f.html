Development Environment

This part of the documentation gives you guidance on the most common and recommended use cases of the installation, configuration and setup of the UI development toolkit for HTML5 (SAPUI5) development environment.

Note

You can install SAPUI5 on different platforms. The license and maintenance conditions of the respective platforms also apply for SAPUI5. If you use SAPUI5 tools on SAP Cloud Platform, for example, the license and maintenance conditions of the SAP Cloud Platform apply.

Depending on your use case, you can choose one of the following development environments.
Note

You can also build native SAP HANA applications using SAPUI5. For more information, see the SAPUI5 for SAP HANA Development Tutorials in the SAP HANA Developer Guide on SAP Help Portal at http://help.sap.com/hana_platform/ under  Development and Modeling  SAP HANA Developer Guide  Building UIs .
