Summary

You should now be familiar with the Worklist app template.

You have learned how to use the SAP Web IDE to create a simple worklist app from a template and you know where to find the code in the Explored app. Based on the initial app you have seen how easy it can be to generate or download initial code and to extend it according to your own requirements. This tutorial also illustrated how to communicate easily with an OData backend using the v2.ODataModel. Furthermore, it illustrated how to use the MockServer with both generated mock data and more realistic data.
