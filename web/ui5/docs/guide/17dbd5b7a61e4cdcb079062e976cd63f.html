Header Facets

You can use the annotation term UI.HeaderFacets to define which information is displayed in the header. For example, you can now define content blocks with several fields and contact details. In addition, you can apply content blocks for the shipping address, price, and category.

The last header section is filled with the data points of your metadata document (com.sap.vocabularies.UI.v1.DataPoint). The title and value are taken into account.

For information about the various types of header facets supported for the object page, see the sections that follow.
