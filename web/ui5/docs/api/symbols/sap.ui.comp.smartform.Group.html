new sap.ui.comp.smartform.Group(sId?, mSettings?)
Constructor for a new smartform/Group.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • useHorizontalLayout : boolean
    • horizontalLayoutGroupElementMinWidth : int
    • label : string
  • Aggregations
    • groupElements : sap.ui.comp.smartform.GroupElement[] (default)
    • layout : sap.ui.layout.GridData

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string}sId? id for the new control, generated automatically if no id is given
{object}mSettings? initial settings for the new control
