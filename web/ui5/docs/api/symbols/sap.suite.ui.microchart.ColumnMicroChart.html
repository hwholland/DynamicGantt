Class sap.suite.ui.microchart.ColumnMicroChartModule: sap/suite/ui/microchart/ColumnMicroChart

extends Control

Compares different values which are represented as vertical bars. This control replaces the deprecated sap.suite.ui.commons.ColumnMicroChart.


Since: 1.34.

new sap.suite.ui.microchart.ColumnMicroChart(sId?, mSettings?)
Constructor for a new ColumnMicroChart control.

Accepts an object literal mSettings that defines initial property values, aggregated and associated objects as well as event handlers. See sap.ui.base.ManagedObject for a general description of the syntax of the settings object.

The supported settings are:

  • Properties
    • size : sap.m.Size (default: Auto)
    • width : sap.ui.core.CSSSize
    • height : sap.ui.core.CSSSize
    • isResponsive : boolean (default: false)
  • Aggregations
    • columns : sap.suite.ui.microchart.ColumnMicroChartData[] (default)
    • leftTopLabel : sap.suite.ui.microchart.ColumnMicroChartLabel
    • rightTopLabel : sap.suite.ui.microchart.ColumnMicroChartLabel
    • leftBottomLabel : sap.suite.ui.microchart.ColumnMicroChartLabel
    • rightBottomLabel : sap.suite.ui.microchart.ColumnMicroChartLabel
  • Events
    • press : fnListenerFunction or [fnListenerFunction, oListenerObject] or [oData, fnListenerFunction, oListenerObject]

In addition, all settings applicable to the base type sap.ui.core.Control can be used as well.

Parameters:
{string}sId? id for the new control, generated automatically if no id is given
{object}mSettings? initial settings for the new control
