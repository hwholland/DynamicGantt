Enum sap.m.ListModeModule: sap/m/library

Defines the mode of the list.

Field Summary
sap.m.ListMode.DeleteDelete mode (only one list item can be deleted via provided delete button)
sap.m.ListMode.MultiSelectMulti selection mode (more than one list item can be selected).
sap.m.ListMode.NoneDefault mode (no selection).
sap.m.ListMode.SingleSelectRight-positioned single selection mode (only one list item can be selected).
sap.m.ListMode.SingleSelectLeftLeft-positioned single selection mode (only one list item can be selected).
sap.m.ListMode.SingleSelectMasterSelected item is highlighted but no selection control is visible (only one list item can be selected).
