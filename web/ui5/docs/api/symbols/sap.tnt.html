Namespace sap.tntModule: sap/tnt/library

SAPUI5 library with controls specialized for administrative applications.

Namespaces & Classes
NavigationListThe NavigationList control is an interactive control, which provides a choice of different items, ordered as a list.
NavigationListItemThe NavigationListItem control represents an action, which can be selected by the user.
SideNavigationThe SideNavigation control is a container, which consists of flexible and fixed parts on top of each other.
ToolHeaderThe ToolHeader control is a horizontal container that is most commonly used to display buttons, labels, selects and other various input controls.
ToolHeaderUtilitySeparatorThe ToolHeaderUtilitySeparator control is used in the sap.tnt.ToolHeader control to specify where the overflow button is placed.
ToolPageThe ToolPage is a layout control, used to put together the parts of a basic tools app - ToolHeader, SideNavigation and contents area.
