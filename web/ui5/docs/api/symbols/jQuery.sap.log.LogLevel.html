Namespace jQuery.sap.log.LogLevelModule: jquery.sap.global

Enumeration of levels that can be used in a call to jQuery.sap.log.Logger#setLevel(iLevel, sComponent).

Deprecated API:Since 1.1.2. To streamline the Logging API a bit, the separation between Level and LogLevel has been given up. Use the (enriched) enumeration jQuery.sap.log.Level instead.
